Usando Quark CSS Framework

Quark CSS es un framework para CSS ultraliviano, apto para maquetado o desarrollo ágil de sitios web responsivos.
Inspirado en Bootstrap, Materialize, y Chota, busca brindar una solución a aquellos que desean tener lo justo y necesario, ni más ni menos, para montar un sitio web desde un simple archivo html.

SISTEMA DE CUADRÍCULA SIMPLIFICADO

Empleamos un sistema de cuadrícula basado en flexbox y porcentajes. Codeá menos, hacé más.


FUENTES PREDEFINIDAS

Contamos con una pequeña base de datos nativa con fuentes para cubrir necesidades básicas.

Una Sans (Lato), una Serif (Didot), y una Monospace (Courier).


LIVIANO Y ACTUALIZABLE

Olvidate de enlazar a un servidor remoto. El NCS completo pesa poco más de 3kb. Y se optimiza periódicamente.
